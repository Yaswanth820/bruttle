
# Bruttle

A small script to bruteforce pdf, zip, and some popular hashes.


### Usage:
```bash
git clone https://github.com/tamton-aquib/bruttle.git
cd bruttle
pip3 install -r requirements.txt

python3 bruttle.py (zip|hash|pdf) <file> <wordlist>
```
